# Code Documentation

## intToChess
This function is used to display chess pieces instead of numbers to make it easier to read in game.

| Int | Chess piece  | Unicode Character |
|:----|:-------------|:------------------|
| 0   | Empty Space  | '  '              |
| 10  | White King   | 'wK'              |
| 11  | White Queen  | 'wQ'              |
| 12  | White Rook   | 'wR'              |
| 13  | White Bishop | 'wB'              |
| 14  | White Knight | 'wK'              |
| 15  | White Pawn   | 'wP'              |
| 20  | Black King   | 'bK'              |
| 21  | Black Queen  | 'bQ'              |
| 22  | Black Rook   | 'bR'              |
| 23  | Black Bishop | 'bB'              |
| 24  | Black Kinght | 'bK'              |
| 25  | Black Pawn   | 'bP'              |
